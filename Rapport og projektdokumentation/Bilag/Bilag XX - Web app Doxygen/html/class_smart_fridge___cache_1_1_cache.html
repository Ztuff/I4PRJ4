Detailed Description

Cache opgave er at indeholde den data der går igen for hvert view og indeholder derfor kun statiske attributter. Dette er foruden alle de pågældende ListItems,List og Items tilkoblet denne liste, også en statisk udgave af ISmartFridgeDALFacade som samtlige controllers skal gøre brug af, for at få adgang til databasen.
